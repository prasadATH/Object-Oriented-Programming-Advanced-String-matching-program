4.6.3RegularExpressions

JavahasaPatternclassforperformingregularexpression(regex)matching.(Thisclasshasnothingwhatsoevertodowiththedesignpatternswevecovered!Itjusthappenstobeusefulinthiscontext.)

Itisntnecessarytounderstandregularexpressionsindetail,forthepurposeofthisassignment,becausetheuserisresponsibleforcreatingthem.However,youwillneedtounderstandthegeneralidea,andbeabletotestthem.Aregularexpressionisastringcontainingplaintextaswellascertainspecialsymbols,whichletitmatchotherstrings:

  • .meansanysinglecharacter;
  • *meanszero-or-moreoftheprecedingitem(sothat.*meansanysequenceofcharacters).
  • +meansone-or-moreoftheprecedingitem;
  • ?meanszero-or-oneoftheprecedingitem;
  • (...)groupsasetofcharacters(sothat(abc)*meanszero-or-moreoccurrencesofabc);
  • [...]meansanysinglecharacterfromthoseinsidethebrackets;
  • ^meansthestartofthetext(sothat^abcmeansabc,butonlyatthebeginningofthestring);
  • $meanstheendofthetext(sothatabc$meansabc,butonlyattheendofthestring);
  • \istheescapecharacter,whichyoucanputbeforeaspecialcharacterlike.,*,\itself,etc.toremoveitsspecialmeaning(sothat\*refersjusttotheliteralcharacter*,nottozero-or-moreofanything);
  • otherstoo,whichyoucanlookupforyourself.

YoucanusethePatternclassasfollows,givenboththeregularexpressionstringandthestringyouwanttomatchitagainst:

  1. ObtainaPatternobjectbycallingPattern.compile(),andpassingtheregularexpressionstringasaparameter.

  2. ObtainaMatcherobjectbycallingmatcher()onthePatternobject,passingthestringyouwanttomatchtheregularexpressionagainstasaparameter.

  3. CalltheMatcherobjectsfind() method.Itwillreturntrueiftheregularexpressionmatchessomepartofthetext,andfalseotherwise.

Forillustration:

import java.util.regex.Pattern;
import java.util.regex.Matcher;
...
String regex = "(abc)*";
String text  = "Some text abcabcabc some text";

Pattern p = Pattern.compile(regex);
Matcher m = p.matcher(text);
if(m.find())
{
    System.out.println("Match found");
}

Important

Therearewaysforregularexpressionstobeinvalid,suchasmismatching()brackets.
